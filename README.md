# 🧩 Modular ETL Pipeline (Python)

## 📌 Overview
This project is a **modular ETL (Extract, Transform, Load) pipeline** built using Python.  
It is designed with a clean architecture to handle data ingestion, validation, cleaning, transformation, and logging in a scalable way.

---

## ⚙️ Features
- 📥 Config-driven pipeline using YAML
- 🧹 Data cleaning (missing values, duplicates)
- ✅ Data validation layer
- 🔄 Data transformation (scaling using StandardScaler)
- 🪵 Logging system for tracking execution
- 🧩 Modular architecture (separation of concerns)
- 📊 Supports multiple datasets dynamically
